# Project Title: ASOIAF Encyclopedia 

## Project Description
This is a landing page for an encyclopedia for all things *A Song of Ice and FIre*.

## Intended User or Player
This is for newcomers from HOTD and AKOTSK interested in the settings
of Westeros and Essos setting and its source material, as well as 
long-time fans well-versed in the lore.

## User / Player Goal
The user should be able to comfortably explore the world, without
having to have a particular article or page in mind.

## Inspiration Interfaces
1. The Civiliopedia
https://www.civilopedia.net/en-US/gathering-storm/concepts/intro/​
Its design is friendly for those who know nothing about the game, with which is perfect for a title that has such huge swaths of information, like ASOIAF.​
Features a quaint hero space that doesn't distract attention away from the plethora of buttons.

2. Witcher 3 Glossary
https://www.youtube.com/watch?v=aQidy_CD-z0​
Its information architecture is perfect for compiling information regarding that of a fantasy world. 
Prominent side bar, and images accompanying the text of the interface's content.
Coloring makes the text easy to read and stand out. 

## Design Choices
- Layout: Like the Civilopedia, it has a condensed interface card with a side bar that has some prominence. The Hero Page serves to immediately let the user know where they are, and the cards below help to highlight certain categories, with a CTA link down below to a particular article.
- Color: Cool colors for the background, which make the golden text stand out, as well as the responsiveness that you get from hover interactions.
- Typography: Took cues from Elden Ring and other medieval video game titles and IPs by using Cinzel, and then Inter for the rest. 
- Visual hierarchy: Hero Card -> Card Group -> CTA -> Side Bar
- Feedback states: Hover states for the side bar that change color and push the text, as well as an added glow for clicking inside of the search bar. 
- Responsive design: The side bar is put on top of the Hero Page in mobile view, and the card group is vertically stacked.
